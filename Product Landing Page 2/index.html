Introduction to HTML

HTML stands for HyperText Markup Language, and it's the backbone of every web page on the internet.

It provides the structure for content, allowing browsers to render text, images, links, and more.

With HTML, developers define sections, headings, paragraphs, and other components of a webpage.

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <p>Welcome!</p> </body> </html>

The code above is the basic skeleton of an HTML document.

Basic Tags

HTML tags usually come in pairs: an opening tag and a closing tag.

Some commonly used tags include <p> for paragraphs and <h1> to <h6> for headings.

Tags define the meaning and structure of your content.

<h1>Main Title</h1> <p>This is a paragraph below the title.</p>
  • <div> - Container for elements
  • <span> - Inline container
  • <br> - Line break
Attributes and Elements

HTML elements can have attributes that provide additional information.

Attributes appear inside the opening tag and are written as name/value pairs.

For example, the href attribute in an anchor tag defines the link's destination.

<a href="https://example.com">Visit Example</a>

Some common attributes include:

  • id
  • class
  • src
  • alt
  • style
Lists in HTML

HTML supports ordered and unordered lists to organize information.

Ordered lists use numbers, while unordered lists use bullets.

<ul> <li>Item One</li> <li>Item Two</li> </ul>

Lists are great for navigation menus, feature lists, and more.

You can also nest lists for more complex structures.

Forms and Inputs

Forms allow users to submit data to a server.

HTML provides several input types like text, email, password, checkbox, and more.

<form action="/submit" method="post"> <input type="text" name="username" /> <input type="submit" value="Send" /> </form>

Labels should be used to describe input fields for accessibility.

You can group related fields using the <fieldset> tag for clarity.
